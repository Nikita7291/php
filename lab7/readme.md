Класс User (Базовый класс) Расположение: src/Classes/User.php Назначение: Служит основой для создания объектов пользователей в приложении. Свойства:

public string $name — имя участника системы.

public string $login — уникальный идентификатор для авторизации.

private string $password — конфиденциальный пароль (ограниченный доступ). Методы:

Конструктор __construct(string $name, string $login, string $password)

Устанавливает начальные значения свойств при инстанцировании класса.

Принимает три обязательных аргумента: имя, логин и пароль.

showInfo(): string

Формирует и возвращает текстовую строку в формате HTML с данными профиля.

Выводит имя и логин в виде стилизованного блока.

Деструктор __destruct()

Выполняется автоматически в момент уничтожения объекта.

Генерирует сообщение: «Пользователь [логин] удален».

Класс SuperUser (Производный класс) Расположение: src/Classes/SuperUser.php Наследование: Является потомком (расширяет функционал) класса User. Назначение: Предназначен для работы с аккаунтами, обладающими особыми полномочиями. Дополнительное свойство:

public string $role — уровень доступа или статус пользователя (например, «администратор»). Методы:

Конструктор __construct(string $name, string $login, string $password, string $role)

Оперирует четырьмя параметрами (базовые данные + роль).

Передает первые три значения в родительский конструктор.

Определяет значение для специфического свойства role.

showInfo(): string (переопределение метода)

Возвращает расширенную HTML-карточку, включающую информацию о правах доступа.

Отображает имя, логин и текущую роль в едином блоке.
